# Spårfräsning

Spår kan bearbetas med hjälp av CAD/CAM-konturer, fickor, eller specialiserade spårfräsningsfunktioner.

Regler för Spårfräsning  
Använd ett verktyg som är mindre än bredden av spåret när så är möjligt.  
En rampingång så som visas i figuren är det mest effektiva sättet att fräsa ett spår.
